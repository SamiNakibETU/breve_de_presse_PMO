# Couleur

## Intentions

- **Mode par défaut** : clair, papier, faible contraste décoratif mais **contraste texte/ fond lisible** (viser WCAG AA sur le corps).
- **Une couleur d’accent** pour l’éditorial : lien actif, rubrique courante, élément sélectionné — **pas** comme décoration répétée.
- **Rouge / live** : réservé au **direct**, à l’urgence réelle, pour ne pas banaliser l’alerte.

## Sémantique

| Token (exemple) | Usage |
|-----------------|--------|
| `bg` | Fond page, ton chaud papier |
| `bg-elevated` | Surfaces qui se détachent légèrement (header sticky, panneau) |
| `text` | Texte principal |
| `text-secondary` | Sous-titres, chapô secondaire |
| `text-tertiary` | Métadonnées, légendes |
| `border` | Filets 1px, séparateurs |
| `accent` | Navigation active, lien fort thématique |
| `live` | Badge direct, filet ou point d’état live |
| `cta` | Bouton d’abonnement / conversion — **visible mais pas « startup »** |

## Interdit ou à limiter

- Gradients de fond sur les articles
- Fonds noirs saturés par défaut
- Multiples couleurs d’accent différentes sur une même vue

## Mode sombre

Optionnel. Si activé : ne pas seulement inverser ; revoir les niveaux de gris et l’accent pour garder le calme éditorial.
